# Git Commit Analyzer

Tool that analyzes git repos for commit patterns, contributor stats, and code velocity metrics.

## Tech Stack

- Node.js
- JavaScript
- Automation

## Getting Started

```bash
# Clone the repo
git clone https://github.com/JASSBR/git-commit-analyzer.git
cd git-commit-analyzer

# Install dependencies (if any)
npm install

# Run the project
npm start
```
